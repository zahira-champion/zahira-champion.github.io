What are the Differences Between Relative, Absolute and Fixed Positioning?

First off lets talk about positioning in HTML. Lets imagine you have inserted a square on your webpage, it will automatically be placed on the page as position: static; in another words flow with the page. We can see this in the example below. This is the default for the HTML property position. The position property is made up of top, right, bottom and left. 

<div class="container">
    <div class="square1"></div>
</div>
                        
.container {
    height: 100px;
    width: 300px;
    border: 0px;
    background-color: white;
}  

.square1 {
    position: static;
    height: 50px;
    width: 50px;
    border: 0px;
    background-color: black;
}

Relative

Now if we wanted to move the element using top, left, botom or right we would use position: relative;. Here i've used left: 30px; and top: 30px;. It has moved it 30 pixels down from the top and pushed it 30 pixels to the right from the left. This is not using the positions of the nearest parent though. 

<div class="container">
<div class="square2"></div>
</div>
                        
.container {
    height: 100px;
    width: 300px;
    border: 0px;
    background-color: white;
}  

.square2 {
    position: relative;
    left: 30px;
    top: 30px;
    height: 50px;
    width: 50px;
    border: 0px;
    background-color: black;
  }

Absolute

The use position: absolute; lets you place the element anywhere on the page. You have to be careful with overusing it, as it limits the flexibility of the page as the elements using this now have no flow on the page. The same positioning attributes top, left, bottom and right are used. Absolute positions it to the nearest positioned parent NOT using static. For example the black box has position: absolute; left: 1px; top: 1px;. The red square has position: absolute; right: 5px; top: 30px;. 

<div class="container3">
<div class="square3"></div>
<div class="red-square"></div>
</div>
                        
.container3 {
position: relative;
height: 100px;
width: 300px;
border: 0px;
background-color: white;
}
                            
.square3 {
position: absolute;
left: 1px;
top: 1px;
height: 50px;
width: 50px;
border: 0px;
background-color: black;
}
                            
.red-square {
position: absolute;
right: 5px;
top: 30px;
height: 50px;
width: 50px;
border: 0px;
background-color: red;
}

Fixed Positioning

Now onto the last one and I say easiest one to understand. position: fixed; is used to keep something fixed to the screen say for example a navagation bar. You don't want to have to scroll all the way back to the top once you have finished reading a long page. You can see an example of this on this page now. As you scroll down the page it always stays at the top of the window and doesn't disappear.
